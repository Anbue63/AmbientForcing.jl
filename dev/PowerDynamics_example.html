It is also possible to perturb the variable using different distributions: Typically SNBS the angle θ and ω are pertubed differently

idx = idx_exclusive(rpg, ["u_r_2", "u_i_2", "θ_2", "ω_2"])
τ = 2.0 # the integration time
2.0

When you want to sample eg. the angle θ from a box of [0, 2π] You have to make sure to devide the distribution argument by the integration time τ

dist_vec = [[0,1], [0, 2], [0, 2π] ./τ, [-5, 5] ./τ]

Then everything works as beforehand. And we see again that we have created a valid initial condition!
